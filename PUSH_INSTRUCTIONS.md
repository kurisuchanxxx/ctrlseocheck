
# Istruzioni per push su GitHub:

1. Crea un nuovo repository su GitHub (https://github.com/new)
   - Nome suggerito: ctrlseocheck o seo-audit-app
   - Scegli 'Public' se vuoi renderlo pubblico
   - NON inizializzare con README, .gitignore o license

2. Una volta creato, GitHub ti mostrerà i comandi. Esegui:

   git remote add origin https://github.com/TUO_USERNAME/NOME_REPO.git
   git branch -M main
   git push -u origin main

Oppure se preferisci SSH:
   git remote add origin git@github.com:TUO_USERNAME/NOME_REPO.git
   git branch -M main
   git push -u origin main

✅ Il file .env è già escluso dal repository (verificato)
✅ Il file .env.example è incluso come template

# Prompt per ChatGPT - Post LinkedIn SEO Audit App

Copia e incolla questo prompt in ChatGPT per generare un post LinkedIn professionale:

---

**PROMPT:**

Ciao! Ho bisogno che tu generi un post LinkedIn professionale e coinvolgente per presentare un'applicazione web che abbiamo appena sviluppato. Ecco tutti i dettagli:

**CONTESTO:**
Abbiamo sviluppato **CtrlSEOCheck**, un tool professionale per analisi SEO automatica di siti web, progettato specificamente per PMI (Piccole e Medie Imprese) locali italiane. L'app è stata sviluppata da **Ctrl Studio**.

**CARATTERISTICHE PRINCIPALI:**

1. **Analisi SEO Completa** (5 aree):
   - **Technical SEO** (25 punti): SSL, velocità, mobile-friendliness, sitemap, robots.txt, link rotti
   - **On-Page SEO** (25 punti): Meta tags, heading structure, immagini, contenuti, Open Graph, Schema markup
   - **Local SEO** (20 punti): Dati NAP (Name, Address, Phone), consistenza NAP, schema markup locale, menzioni località, Google Business Profile
   - **Off-Page SEO** (10 punti): Backlinks stimati, directory listings, domain authority
   - **Analisi Competitor**: Confronto comparativo con fino a 3 competitor

2. **Integrazione PageSpeed Insights API** (FIRST-MOVER):
   - Metriche reali Google API per Performance, Accessibility, Best Practices, SEO
   - Core Web Vitals: LCP (Largest Contentful Paint), CLS (Cumulative Layout Shift), TBT (Total Blocking Time), FCP (First Contentful Paint)
   - Analisi separata per dispositivi mobili e desktop (come PageSpeed Insights ufficiale)
   - Ottimizzazioni risorse (render-blocking, compressione, immagini)

3. **AEO/RAO - Answer Engine Optimization** (FIRST-MOVER - caratteristica esclusiva):
   - Analisi per ottimizzare contenuti per AI e motori di risposta (ChatGPT, Claude, Perplexity, Google AI Overview)
   - Struttura Q&A (domanda-risposta) per AI
   - Schema Markup AEO: FAQ, HowTo, Article per motori di risposta
   - Contenuti citabili: statistiche, fonti, paragrafi snippet-ready
   - Ottimizzazione semantica: topic depth, link interni, domande correlate
   - Formato e leggibilità: analisi lunghezza frasi, paragrafi, keyword evidenziate
   - Autorevolezza: valutazione contenuti per citazioni AI

4. **Dashboard e Report Avanzati**:
   - Sistema di scoring 0-100 con breakdown per categoria
   - Raccomandazioni dettagliate con:
     * Esempi di codice HTML/CSS/JS
     * Link a risorse utili
     * Metriche specifiche (attuale vs target)
     * Stima tempo implementazione e difficoltà
   - Report PDF professionale brandizzato Ctrl Studio
   - Storico analisi: salvataggio e confronto evolutivo
   - Quick Wins: sezione dedicata per miglioramenti rapidi ad alto impatto

**STACK TECNOLOGICO:**
- **Frontend**: React 18+ + TypeScript + Tailwind CSS + Recharts (grafici) + jsPDF (generazione PDF)
- **Backend**: Node.js + Express + TypeScript + Cheerio (web scraping) + SQLite (database)
- **API**: Google PageSpeed Insights API v5
- **Build Tools**: Vite, Nodemon, TypeScript

**PUNTI DI FORZA UNICI:**
1. **FIRST-MOVER in AEO/RAO**: Primo tool che include analisi Answer Engine Optimization per ottimizzare contenuti per AI e motori di risposta
2. **Integrazione PageSpeed Insights**: Metriche reali Google con Core Web Vitals separati per mobile e desktop
3. **Focus PMI Locali Italiane**: Ottimizzato per business locali con analisi Local SEO specifica
4. **Raccomandazioni Actionable**: Non solo problemi, ma soluzioni concrete con esempi di codice e stime di tempo
5. **Report PDF Brandizzato**: Export professionale pronto per clienti

**TARGET:**
PMI locali italiane (ristoranti, hotel, artigiani, professionisti) che vogliono migliorare la loro presenza online e SEO.

**TONO DEL POST:**
- Professionale ma accessibile
- Enfatizza l'innovazione (AEO/RAO come FIRST-MOVER)
- Evidenzia il valore pratico per le PMI
- Include call-to-action per provare l'app
- Usa emoji con moderazione (max 3-4)
- Lunghezza: 2000-2500 caratteri (post LinkedIn ottimale)

**STRUTTURA SUGGERITA:**
1. Hook iniziale che cattura l'attenzione (problema comune delle PMI con SEO)
2. Presentazione della soluzione (CtrlSEOCheck)
3. Caratteristiche principali (con focus su AEO/RAO come innovazione)
4. Benefici pratici per le PMI
5. Call-to-action

**REQUISITI:**
- Post in italiano
- Linguaggio professionale ma non troppo tecnico
- Evidenzia l'aspetto FIRST-MOVER per AEO/RAO
- Menziona Ctrl Studio come sviluppatore
- Include hashtag rilevanti (max 5-7)
- Non troppo promozionale, più informativo/educativo

Genera un post LinkedIn che rispetti questi requisiti e che sia coinvolgente e professionale!

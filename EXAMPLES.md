# Esempi di Utilizzo - SEO Audit Tool

## Esempio 1: Ristorante Locale

### Input
- **URL**: `https://ristorante-esempio.it`
- **Settore**: Ristorante
- **Località**: Milano, Lombardia
- **Competitor**: 
  - `https://competitor1-ristorante.it`
  - `https://competitor2-ristorante.it`

### Output Atteso
- **Score**: 65-85/100 (dipende dalla qualità del sito)
- **Technical SEO**: Verifica SSL, velocità, mobile
- **On-Page SEO**: Meta tags menu, immagini piatti con alt text
- **Local SEO**: NAP nel footer, schema LocalBusiness, menzioni "Milano"
- **Raccomandazioni**: 
  - "Aggiungi schema LocalBusiness con orari apertura"
  - "Ottimizza immagini menu per velocità"
  - "Crea pagina dedicata a Milano"

## Esempio 2: Hotel/B&B

### Input
- **URL**: `https://hotel-esempio.com`
- **Settore**: Hotel
- **Località**: Firenze, Toscana
- **Competitor**: 
  - `https://competitor-hotel.it`

### Output Atteso
- **Score**: 70-90/100
- **Technical SEO**: Verifica booking system, velocità pagine
- **On-Page SEO**: Meta tags per ogni camera, immagini strutture
- **Local SEO**: Indirizzo completo, coordinate GPS, recensioni
- **Raccomandazioni**:
  - "Aggiungi schema Hotel con rating e prezzo"
  - "Ottimizza immagini camere (WebP)"
  - "Linka Google Business Profile"

## Esempio 3: Artigiano/Professionista

### Input
- **URL**: `https://artigiano-esempio.it`
- **Settore**: Artigiano
- **Località**: Roma, Lazio
- **Competitor**: Nessuno

### Output Atteso
- **Score**: 50-75/100 (siti artigiani spesso meno ottimizzati)
- **Technical SEO**: Verifica portfolio, velocità immagini
- **On-Page SEO**: Meta tags servizi, portfolio con alt text
- **Local SEO**: Zona servita, recapiti chiari
- **Raccomandazioni**:
  - "Crea pagine dedicate per ogni zona servita"
  - "Aggiungi schema LocalBusiness con servizi"
  - "Ottimizza immagini portfolio"

## Note Sugli Esempi

⚠️ **Importante**: 
- Sostituisci gli URL di esempio con siti reali e raggiungibili
- Alcuni siti potrebbero bloccare richieste automatizzate
- I punteggi sono indicativi e dipendono dalla qualità reale del sito
- Per test completi, usa siti che conosci e di cui puoi verificare i risultati

## Interpretazione Risultati

### Score 80-100
Sito ben ottimizzato, poche azioni necessarie. Focus su miglioramenti incrementali.

### Score 60-79
Sito con buone basi ma spazio di miglioramento. Priorità su quick wins.

### Score 40-59
Sito con problemi significativi. Necessaria ottimizzazione strutturale.

### Score 0-39
Sito con problemi critici. Richiede intervento completo.

## Quick Wins Comuni

1. **Aggiungi meta description** (5 minuti, alto impatto)
2. **Ottimizza immagini** (30 minuti, alto impatto)
3. **Aggiungi schema LocalBusiness** (15 minuti, medio impatto)
4. **Correggi dati NAP** (10 minuti, alto impatto per local SEO)
5. **Crea sitemap.xml** (5 minuti, medio impatto)
